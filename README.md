# 🐱 Meow Error

Meow Error plays a cute meow sound whenever a code error occurs in VS Code.

## 🚀 Features

- Detects real-time syntax errors
- Plays a meow sound instantly
- Cooldown protection (prevents repeated sound spam)

## 📦 Installation

Search for **Meow Error** in the VS Code Extensions Marketplace and click Install.

## 🛠 How It Works

The extension listens for diagnostics changes and triggers a sound whenever errors are detected.
